    How to Play

    1. 1Read the Java code. Every token is clickable.

    3. 3Pick a label that describes what the selected code does -- data type, variable name, operator, etc.
    4. 4Your tag appears in the list below. You can tag the same code multiple ways -- for example, int is a "data type" and int score together is a "variable declaration".
