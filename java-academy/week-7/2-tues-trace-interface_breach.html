SECTION 1 // THE INTERFACES
Q1 //
Clearable.java and Loggable.java -- interface declarations
Both interfaces list method signatures with no bodies. When a class uses the implements keyword with one of these, what is it promising Java? What does Java do at compile time if that promise is not kept?
Q2 //
Agent.java -- class declaration, line 1
Agent implements two interfaces. What does Java require Agent to provide because of this? What happens at compile time if one method is missing?
