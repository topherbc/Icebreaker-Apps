    How to Play

    1. 1Select a problem from the list on the left to load it into the exercise area.
    2. 2Read the problem statement. Click any highlighted phrase to open the tagging panel.
    3. 3Select one or more tags that describe what the phrase implies about the code — pick everything that applies.
    4. 4Click Add tag to save. Tag as many phrases as you like — your tags appear below the statement as you go.
    5. 5When you're ready, click Check my work to compare your annotations against the expected answers.
    6. 6Use Reveal code to see the full Java solution for the problem.
    7. 7Use the Export for AI feedback to see have a deeper analysis of the problem statement - and what can be improved
